# Joshua Lenz Portfolio Website

## Project Overview

This project is a website built with HTML, CSS, and JavaScript. The site presents my background as a computer science grad. The purpose fo this website was to act as a portfolio for my previous work, projects, and a way to contact me. 

## Live Site

The site is deployed using GitHub Pages:

https://tikipalooza.github.io/cs463-final-portfolio/

## Features

- Single-page layout with anchor navigation
- Sticky navigation header
- Hero section with resume download links
- About section using responsive CSS Grid
- Previous work section with featured and standard work cards
- Projects section with category filter buttons
- Contact form with client-side validation
- Responsive layouts using media queries
- Custom dark theme using CSS custom properties
- Hover and focus states for interactive elements

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap

> Bootstrap was used only for the `.container` layout utility. The rest of the site styling was written with custom CSS.
